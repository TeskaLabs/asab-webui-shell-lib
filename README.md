# TeskaLabs ASAB WebUI Shell Library

Shell library of ASAB used in WebUI projects.

## Getting started

```
pnpm add asab_webui_shell
```

## Usage

```
import { Application } from 'asab_webui_shell';
```

Possible imports:

- Application
- AboutModule
- AuthModule
- TenantModule
- I18nModule
- getBrandImage
